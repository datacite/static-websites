DataCite Developer Machine

This machine runs several services used for testing, building and monitoring DataCite's infrastructure.

Visit DataCite on GitHub for source code or issues.

If you have any questions please contact us via tech@datacite.org.

Test Instances

We have set up instance of all our service for testing purposes. This is a closed system, so e.g. minted DOIs show only up in the test handle server, uploaded metadata only in the test search instance, etc.

Metadata Store

This is a development version of MDS for testing new features in MDS. It is configured as close as possible to our production version of MDS, but there are some differences:
  • Metadata: The namespace and schemaLocation in your xml have to start http://schema.datacite.org/meta/. However the metadata is validated via our test schema instance. So you can even pick a non-released schema version.
  • E-Mails are currently disabled. If you have forgotton your password, please ask us to reset it.
If you need an account please send a e-mail to tech@datacite.org.
